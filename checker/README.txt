========================
UTILISATION DU CHECKER
========================

1. Format du fichier d'instance

Il est imperatif que le fichier d'instance respecte le format proposé dans la description du sujet.



2. Format du fichier de solution

Il est imperatif que le fichier de solution respecte le format proposé dans la description du sujet.



3. Nommage des fichiers

Si le fichier d'instance se nomme 'nomInstance.txt', alors le fichier de solution doit se nommer 'nomInstance_sol.txt'.



4. Utilisation du checker pour une instance

- Dans un meme repertoire mettez le checker (CheckerBatchingPicking.jar), l'instance et la solution avec les bons formats.
- Ouvrez une fenêtre de ligne de commande
- Tapez
	java -jar CheckerBatchingPicking.jar nomInstance
	
Notez bien que dans ce cas, il est impératif que le fichier d'instance se nomme 'nomInstance.txt', et le fichier de solution se nomme 'nomInstance_sol.txt'.



5. Utilisation du checker pour plusieurs instances

- Dans un meme repertoire mettez le checker (CheckerBatchingPicking.jar), les instances et les solution avec les bons formats.
- Ouvrez une fenêtre de ligne de commande
- Tapez
	java -jar CheckerBatchingPicking.jar -d
